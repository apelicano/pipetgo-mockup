Component-Based Architecture

The application is built using a modular, component-based structure. This approach ensures reusability, consistency, and maintainability across all user roles, including logistics management.

Atoms: The Building Blocks

Button
Input
Label
Icon (e.g., Unicode)

Molecules: Simple Components

Search Bar (Input + Button)
Service Card (Info + Actions)
Logistics Task Card (Order Info + Status Update)

Organisms: Complex Sections

Header (Logo + Nav + Profile)
Order Tracking View (Details + Logistics Status + Lab Status)
Logistics Dashboard (Pickup List + Map View Placeholder)
